# Arduino  UNO 编程积木

NyBoard相当于一块通用的Arduino Uno板，具有丰富的外设。 除了原生 Arduino IDE 之外，您还可以使用 Mind+ 积木块对其进行编程。 但要注意，如果使用此模式，原有的 OpenCat 固件会被覆盖，稍后需要重新上传固件才能恢复默认的机器人功能。
